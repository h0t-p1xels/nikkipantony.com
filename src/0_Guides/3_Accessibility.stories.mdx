## Design System Keyboard Shortcuts

- Show/hide navigation sidebar <kbd>S</kbd>
- Show/hide top toolbar <kbd>T</kbd>
- Show/hide component canvas addons bar <kbd>A</kbd>
- Change orientation of component canvas addons bar <kbd>D</kbd>
- Search components <kbd>/</kbd>
- Previous page/component <kbd>alt</kbd> + <kbd>&#60;</kbd>
- Next page/component <kbd>alt</kbd> + <kbd>&#62;</kbd>
- Zoom In <kbd>alt</kbd> + <kbd>↑</kbd>
- Zoom Out <kbd>alt</kbd> + <kbd>↓</kbd>
